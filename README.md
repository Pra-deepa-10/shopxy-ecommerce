# shopxy-ecommerce
Modern Ecommerce Store built with React featuring product search, shopping cart management, quantity controls, routing, and responsive UI.

🚀 Features

✔ Product Listing
✔ Shopping Cart
✔ Add / Remove Products
✔ Quantity Management
✔ Product Search
✔ React Router Navigation
✔ Responsive UI
✔ Modern Dark Theme

🛠 Built With

- React
- React Router
- CSS3
- JavaScript ES6
